# 🧠 NeuroTrace

**Passive neurological risk detection from typing behaviour.**

NeuroTrace is a dual-stream behavioral classifier that analyses keystroke timing and language patterns to output a neurological risk profile across three classes: Parkinson's Disease, MCI/Alzheimer's, or Healthy baseline.

---

## Quick Start (Demo Mode — No Datasets Required)

```bash
# 1. Create and activate virtual environment
python -m venv neurotrace-env
# Windows:
neurotrace-env\Scripts\activate
# Mac/Linux:
# source neurotrace-env/bin/activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Generate synthetic data + train all models
python train_all.py

# 4. Launch the demo UI
streamlit run app/demo.py
# Opens at http://localhost:8501
```

## Using Real Datasets

### neuroQWERTY (Parkinson's keystroke data)
```bash
# Requires free PhysioNet account: https://physionet.org
pip install wfdb
python -c "import wfdb; wfdb.dl_database('nqmitcsxpd', dl_dir='data/raw/nqmitcsxpd')"
```

### DementiaBank / Pitt Corpus (MCI language transcripts)
1. Register at https://dementia.talkbank.org (free, ~5 min)
2. Download the Pitt Corpus .cha files
3. Place in `data/raw/dementiabank/`

Then re-run `python train_all.py` — the loaders will automatically detect and use real data.

## Architecture

```
Keystroke Log ──→ Motor Features ──→ PD Stream (XGBoost) ──→ P(PD)  ──┐
                                                                          ├──→ Fusion (LogReg) ──→ 3-class Risk
Speech/Text ────→ Linguistic Feats → MCI Stream (XGBoost) ──→ P(MCI) ──┘
                                          ↓
                                   SHAP Explainability
```

## Disclaimer
NeuroTrace is a research prototype and screening tool. It is **not a medical device** and outputs should not replace clinical diagnosis. All predictions include a confidence interval and are intended to prompt — not replace — clinical assessment.
